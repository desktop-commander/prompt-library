üß™ Manual Copy Testing Guide

Test 1: Text Selection
1. Open a prompt modal
2. Highlight some of the prompt text
3. Check PostHog for prompt_text_selected
Test 2: Keyboard Copy
1. Select prompt text
2. Press Ctrl+C (or Cmd+C on Mac)
3. Check PostHog for prompt_manual_copy_attempt with method="keyboard"
Test 3: Right-Click Copy
1. Select prompt text
2. Right-click to open context menu
3. Click "Copy" from menu
4. Check PostHog for prompt_right_click_detected + prompt_manual_copy_attempt
Test 4: Selection Abandonment
1. Select prompt text
2. Click elsewhere to deselect (without copying)
3. Check PostHog for prompt_text_deselected with abandoned_copy_intent=true
Test 5: Full vs Partial Selection
1. Select just a few words
2. Select the entire prompt
3. Compare is_full_prompt_selected values in PostHog
